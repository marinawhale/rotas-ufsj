# 🚌 **COMO CHEGAR NA UFSJ**
**Consulta de horários das linhas universitárias de Ouro Branco de forma simples**
##
**O que o sistema faz**

- Filtro por **bairro**
- Filtro por **ponto de parada**
- Lista completa de horários
- Destaque do **próximo ônibus**
- Atualização automática ao selecionar filtros

---

## **Como os dados funcionam**

Os horários são organizados em um JSON simples:

    {
      "bairro": "Centro",
      "ponto": "Rodoviária",
      "horarios": ["11:10", "12:10", "14:10"]
    }
